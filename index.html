ObjViewer - Winter 2016

Learning to render 3D with OpenGL from scratch

Description

The only libraries used are win32 and OpenGL. It renders an .obj file passed in through the command line.

Links

GitHub repository
Download executable
Example .obj files

Instructions

Run as "ObjViewer.exe obj_file"

WASD to translate, QE to rotate (x/y/z to change axis of rotation), FR to zoom in / out, hold shift to speed up any of those actions.

If you don't see the object try zooming out with R.

Project Ibix - Fall 2013

Made with GameMaker Studio for a first-year course

Description

It's a 2D arcade-style action game with three weapons, four enemies, and 15 levels. Unfortunately I seem to have lost the code for it! Credits are in-game.

Links

Download executable
Play online (very buggy - strongly recommend executable; works better in IE than chrome)

Instructions

Controls and tutorial are in-game. Press ESC to open pause menu, WASD to move, left/right mouse buttons and LShift to use weapons, Space to jump, W to dash.

It is strongly recommended to play on Casual difficulty and use the "Auto select" feature when upgrading. It's not balanced well.

Unity "tech demo" compilation - Fall 2014 / Winter 2015

Early studies in Unity

Description

I've fiddled around with Unity a few times and made some different "tech demo" projects. They're all pretty unimpressive on their own, so I took the important features from each one and combined them into this. The most interesting feature is the enemy pathfinding, using the A* pathfinding project.

Links

GitHub repository
Play online (works better in IE than Chrome)
Download executable

Instructions

Controls are in-game. Enemies move by cycling through fixed waypoints (unique for each enemy) - try placing walls in their paths and watch them path around them.
